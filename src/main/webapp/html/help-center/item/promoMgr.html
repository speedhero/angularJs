一、促销价格设置

1、 设置促销价格

1.1 进入飞飞厂商后台，【促销】【促销价列表】，【添加商品促销价】

1.2 设置促销价格生效时间和失效时间；

1.3 挑选需要参加促销调价商品；可通过条件筛选查询，也可以全选全部添加或单独添加进入调价列表；

注意：只有上架的商品，才能参加促销调价；

1.4 挑选完商品之后，对于挑选错误的商品，可删除；确认无误之后，点击【确定】；

1.5 填写促销价格；确认无误之后，点击【保存】；

注意：一旦保存，即不可修改；请确认无误；

若设置错误价格，可进行【失效处理】；

2、 查看促销列表

2.1促销列表分为三个状态：

  • 【未生效】：未到商品促销价格生效时间；
  • 【已生效】：时间在商品促销价格生效时间；活动价格正在生效；
  • 【已失效】：商品促销价格已过期；

2.2 设置价格失效；

若设置错促销价格，点击【设置失效】，将取消该商品促销价格；若需重新设置，请重新操作【设置促销价格】

二、店铺购物车规则

1、 设置店铺优惠列表

进入飞飞厂商后台，开始设置店铺优惠；

【一】、创建全店活动；

1.1 全店满**元即送赠品

1.1.1 填写活动名称，名称描述，设置生效和失效时间，确定优惠条件等；

注意：活动名称将会在商城前端展示，请严谨命名；

1.1.2 挑选赠品；

注意：

  • 赠品必须已经上架的商品；
  • 设置赠品之后，不会影响作为赠品的商品正常销售；
  • 最多可设置三个赠品，即买一送三；如：买吊灯送灯泡、小台灯、阳台灯；

1.1.3 确认无误，点击【保存】；

生效时，活动展示效果如下：

1.2 全店上不封顶满减；

1.2.1 填写活动名称，名称描述，设置生效和失效时间，确定优惠条件等；

注意：活动名称将会在商城前端展示，请严谨命名；

1.2.2 确认无误之后，点击【保存】

1.3 全场设置梯度满减

1.3.1填写活动名称，名称描述，设置生效和失效时间，确定优惠条件；

注意：活动名称将会在商城前端展示，请严谨命名；

1.3.2 确认无误之后，点击【保存】

【二】、创建部分商品活动；

部分商品活动规则如同全店活动；

需要添加参加活动商品（每个活动参加商品个数上限为100个）：

注意：每个商品只能只能参加一个活动；即A商品参加了全店之后，就无法单独设置部分商品活动；或是如果A商品已经设置了部分商品活动，如A商品每满500元送台灯，那么A商品就无法参加全店活动；

2、查看、编辑优惠规则

2.1 在店铺优惠列表中，可查看优惠规则，优惠规则有三种形式：

  • 【未开始】：未到规则生效时间；这时可进行重新修改编辑或删除该规则；
  • 【已开始】：活动正在生效；若规则有误，可进行【撤销】处理；
  • 【已撤销】：活动规则已被撤销；若需生效，需进行重新设置；

三、 设置店铺优惠券

1、进入飞飞厂商后台，添加优惠券；

2、填写优惠券信息：

  • 【面额】：即优惠券可抵消多少金额；
  • 【有效时间】：即优惠券可使用时间；
  • 【数量】：即优惠券总数量，领完即止
  • 【使用条件】：不限：即无门槛优惠券；不管消费多少金额都可使用；满**元：即满**-优惠金额；如以下，即：满30-5元优惠券；

3、确认无误之后，点击【保存】

4、查看优惠券列表；优惠券分为三种状态：

  • 【派发中】：优惠券一旦生成，即自动上架；并且消费者可点击领取；
  • 【已结束】：厂商可进行下架优惠券处理，下架之后，优惠券状态为已结束；
  • 【已失效】：优惠券过了生效时间之后，会自动失效；
